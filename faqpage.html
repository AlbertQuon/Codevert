Frequently Asked Questions

  • What's the difference between the demo and the premium version?

  • The demo version allows limited accessibility - only 15 files can be translated, and only between the following programming languages: Python, Java, Javascript, C, C++. The premium version grants access to courses for learning the supported programming languages, allows user to translate unlimited files, supports librairies, gives user access to 24/7 technical support, and has customized libraries for each language. In addition, the premium version offers the option of downloading the app, which allows offline work.



  • Which operating systems does Codevert support?

  • Windows, Mac OS, and Linux.



  • What languages is the app offered in?

  • English, French, Spanish, Chinese (Traditional), German, Hindi, Korean, Japanese, Russian.



  • Does Codevert work with older versions of Python?

  • Codevert supports all versions of Python released after 2010, so from Python 2.6.5 to 2.7.17 and most versions of Python 3 (only Python 3.0, 3.0.1, 3.1, 3.1.1 not supported).



  • Can I download the app on more than one device?

  • Yes, as long as you are subscribed to the premium plan, the downloads are unlimited.



  • Which browsers does Codevert support?

  • Codevert is compatible with most of the popular browsers, like Google, Edge/Internet Explorer, Firefox, Safari, and Opera. It might work with other browsers, but we don't guarantee it.



  • Do I need to make an account in order to get the premium version?

  • Yes



  • Where is the premium version subscribe option?

  • Click "Plans" from the nav bar.
